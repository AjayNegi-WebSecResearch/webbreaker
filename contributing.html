Contributing to webinspectapi

Issues

Issues are always welcome! You can expect conversation.

Pull Requests

These rules must be followed for any contributions to be merged into master. A Git installation is required.

  1. Fork this repo
  2. Create a branch
  3. Complete desired changes
  4. Validate the changes meet your desired use case
  5. Ensure documentation has been updated
  6. Open a pull-request: you can expect a discussion
